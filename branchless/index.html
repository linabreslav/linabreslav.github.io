Goodreads Redesign

a redesign of the ‘bookshelf’ page at goodreads.com

About
This project was a redesign of the current Goodreads.com Bookshelf page for my TIY class. After some short usability testing, I learned that users found the current site cluttered, outdated, confusing, and unpleasant to use. Bookshelf is a design-only project that emphasizes minimalism, inspiration, and highlights the beauty of a collection of book covers, taking inspiration from the books on your shelf.

Tools
Illustrator Photoshop CSS HTML5 Javascript

Process
1. Short usability test 2. Redesign logo to reflect minimalist, inspirational aesthetic 3. Design sketches, then mockups 4. Responsive code using HTML5, CSS, and Javascript
